# TO-DO(Django)

A simple to-do task  application built with Django.

## Features
- Add and display task
- Edit and delete task
- User authentication


## Tech Stack
- Python
- Django
- SQLite 
- HTML/CSS

## Run the Project

```bash
git clone https://github.com/achintyah9895/TO-DO.git
cd TO-DO
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python manage.py runserver
